# bank-management-system
a bank management system built in java, utilizing mysql 


## Features

User registration and login​

Account creation with unique account numbers​

Deposit and withdraw funds with balance checks​​

View account details and mini statements​



## Technologies

Java (with Swing/AWT GUI)

MySQL database

JDBC for database connectivity


## Getting Started

Clone this repository.

Set up a MySQL database and update credentials in the code.

compile with -> javac -cp ".\src;.\lib\jcalendar-1.4.jar;.\lib\mysql-connector-j-9.5.0.jar" .\src\bank\management\system\\*.java

then run with -> java -cp ".\src;.\lib\jcalendar-1.4.jar;.\lib\mysql-connector-j-9.5.0.jar" bank.management.system.Login
